# **saio-api**

this project inspired by [cobalt]("https://github.com/imputnet/cobalt")

supported service

```
1.tiktok
```

## how to use

install package

```
npm install

```

set the env (you can use any open port)

```
APP_PORT=3003
```

start dev mode

```
npm run dev
```

start project

```
npm run start
```

this app deployed at [**saio-api**]("https://vercel")
